LaTeX configuration for VScode

Posted on Fri, Feb 7, 2020 latex

Download MikTeX from the official site https://miktex.org/2.9/setup. Optionally a distribution such as BasicTeX or TeXLive could be installed (in case of lack of space and different OS), but it is recommended to download the MikTeX if possible. After downloading, perform the installation.

Download and install the Strawberry Perl program from the official site http://strawberryperl.com.

Install the LaTeX Workshop extension from extension marketplace in VSC.

Open the MikTeX Console and navigate to packages. Search for 'latexmk' package, download and install.

Restart the VScode and enable extensions.

Create new file with .tex extension and copy the following code:

\documentclass[12pt,a4paper]{article}
\begin{document}
Hello world.
\end{document}

Via VScode command help choose 'Build LaTeX project' and preview the builded .pdf file.
